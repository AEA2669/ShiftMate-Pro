# Support — ShiftMate Pro

Thanks for using **ShiftMate Pro**! This project is **free and open-source**.

## Get Help

### 1) Check the Basics
Before reporting an issue, please try:
- Updating to the latest release
- Closing and reopening the app
- Restarting your device (if the issue seems “stuck”)

### 2) Report a Bug
If you found a bug, please open a GitHub Issue and include:

- **Device / OS:** (e.g., iPhone 15, iOS 18.2 / Pixel 8, Android 15)
- **App version:** (e.g., 1.0.0)
- **Steps to reproduce:** (1… 2… 3…)
- **Expected behavior:** what you thought should happen
- **Actual behavior:** what happened instead
- **Screenshots / screen recording:** if helpful
- **Logs (optional):** if you have any relevant error messages

> Tip: Clear, repeatable steps = fastest fix.

### 3) Request a Feature
Feature requests are welcome! Please open a GitHub Issue and describe:
- The problem you’re trying to solve
- What you want the app to do
- Any examples or mockups (optional)
